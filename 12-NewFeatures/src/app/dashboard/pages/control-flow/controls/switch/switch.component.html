SWITCH

The syntax for switch is similar to if, inspired by the JavaScript switch statement:

        
        switch (condition) {{'{'}}
          case (caseA) {{'{'}}
            Case A.
          {{'}'}}
          case (caseB) {{'{'}}
            Case B.
          {{'}'}}
          default {{'{'}}
            Default case.
            {{'}'}}
          {{'}'}}
